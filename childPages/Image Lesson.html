Image Lessons

Steps in building an Image on a webpage

  • Pick an Image (careful of the file extension in the URL)
  • Record URL, cite it, comment for data access, and record original dimensions
  • Use blockquote and comment and appropriate
  • Use img-tag with attributes src, alt, title, and style with width-height
